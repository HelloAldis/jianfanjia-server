《暗香》

报价：18万元|风风格：现代|空间：三居|面积：98平米|浏览量：324

案例简介 Case description

此作品《暗香》的设计感整体和谐，色彩以高级灰为主，在客厅的设计中我们可以看见整个空间采用无主灯的设计，简约而不简单。现代简约风格在《暗香》中大放异彩，打破了传统的以单一的色彩为配色系统的局面，餐厅的灯也是一大亮点，卧室没有复杂的设计，整体都是靠色彩在做效果，在空间中以简练和节约为主，尽量采用柔和的灯光，在温馨细腻的简装环境下，感受浪漫的气息。整体设计带来心理上的平和、恬静，追求沉静，舒畅，内敛，含蓄的知性之美。
