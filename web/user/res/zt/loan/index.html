常见问题

Q“诚一贷•装修贷款”是一款什么样的产品？
“诚一贷•装修贷款”属消费类无抵押免担保贷款产品，根据月收入、按揭房月供等依据测算贷款额度。
Q申请“诚一贷•装修贷款”需要什么条件？
只要您是简繁家会员，且在家装方面有真实需求，便可尝试申请。
Q装修贷如何还款？
您好，装修贷支持绑定银行卡自动代扣还款方式。自动代扣需在您申请时候设置开通，扣款卡为当时绑定银行卡。
Q“诚一贷•装修贷款”申请后多久可以获得额度？
申请提交成功后，大概需要一个工作日时间完成审核，这段时间请保持电话畅通，审核结果会通过短信发送到您的手机上。
Q“诚一贷•装修贷款”有哪些亮点？
“诚一贷•装修贷款”区别于传统消费贷款（房贷、车贷、信用卡等），而“诚一贷•装修贷款”充分发挥了普遍有家装需求的“贷款周期灵活、贷款金额高、贷款时间急”等需求特点。贷款额度最高20万元、月供最低364元起、可贷3年、全程在线申请，30分钟内快速审核、无需抵押和担保，“诚一贷•装修贷款”充分发挥了互联网金融——短、平、快的优势。
Q申请“诚一贷•装修贷款”产品需要准备什么资料？
“诚一贷•装修贷款”仅需您提供个人身份证明、房产证明、收入证明即可申请。
Q我如何使用“诚一贷•装修贷款”？
审批通过后的贷款可由你自主选择打入个人账户或商家账户。
Q“诚一贷•装修贷款”的申请受理时间？
装修贷的申请时间为7×24小时，审核在一个工作日完成。
